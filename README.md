# Wirtschaftsinformatik Bachelorarbeit (inkl. Projektdateien)

> Business Informatics Bachelor Thesis (incl. Project Files)

## Getting Started

Clone the repository **with submodules** using

```shell
git clone git@github.com:Niggelgame/thesis-winfo.git --recursive
```

This will download the writeup source, a subset of the reading material and the code used for the thesis project.

## Writeup

The writeup (*the* thesis document) is contained in the [writeup directory](writeup/README.md).

## Project code

The project code and documentation (especially technical details dropped from the writeup) are contained in the [code directory](code/README.md).
